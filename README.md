# Oasis-Apartment
## Project Description
Oasis Apartment is a simple front-end apartment booking website built with HTML, CSS and JavaScript.
It allows users to view apartment information, check room availability, book rooms and manage bookings using browser localStorage.

## Features
Multi-page website
Room availability checking bases on dates
Room booking and cancellation
booking history per room
User booking details display
Persistent data storage using localStorage
Responsive layout
